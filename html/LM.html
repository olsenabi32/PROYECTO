JavaDoc en el Proyecto

Para documentar el código del proyecto, utilicé JavaDoc, que genera automáticamente páginas HTML a partir de los comentarios del código.
Usé etiquetas como @param para describir los parámetros de cada método, y @return para explicar qué devuelve un método. Esto permite tener una documentación clara, navegable y profesional del proyecto.

Creación de Archivos XML

En el proyecto también generé archivos XML para guardar las compras de cada usuario. Cada archivo tiene un nodo raíz <compras>,
y dentro contiene varios nodos <compra> que incluyen la información del usuario, el precio total y los componentes comprados.
